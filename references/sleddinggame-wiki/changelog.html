Sledding Game is in Early Access, so patches and new content can arrive any time — bookmark this page if you like to read release notes when updates drop.

🗓️ Release History

🎉 Sledding Game Early Access Full Launch

Early Access 1.0 • April 30, 2026

release
  • → 12+ playable characters available at launch
  • → 90+ purchasable/unlockable cosmetic items
  • → 20+ sled types including Snowmobile, Speed Sled, and Toboggan
  • → 70+ sled trinket accessories
  • → 20 catchable fish species (Fishing Rod required)
  • → 8 mini-games: Fishing, Darts, Curling, Cornhole, Marshmallow Roasting, Hot Chocolate, Snowball Fight, Snowman Building
  • → 37 Steam Achievements and 37 Xbox Achievements (1,000 Gamerscore)
  • → Xbox Game Pass Day-One launch (Ultimate and PC tiers)
  • → Proximity voice chat supporting up to 50 players per server
  • → Steam Cloud save support
  • → 7 bundle deals including PEAK & Sledding Game Bundle
The full Early Access launch represents a significant content expansion from the Demo. The developer (Max / The Sledding Corporation) has committed to a full release within 12 months.

🛷 Sledding Game Demo Launch — Viral Moment

Demo v1.0 • September 25, 2025

demo
  • → First public playable version of Sledding Game
  • → Core sledding mechanics introduced
  • → Base Lodge area and main slope available
  • → Initial character roster (subset of full game)
  • → Proximity voice chat tested in Demo
  • → Hot Chocolate mini-game available in Demo
  • → Fox and Toad character unlock challenges included in Demo
The Demo launch was a watershed moment: 80,000 Steam wishlists in the first week, 11.4× the developer's expectations. VICE called it 'the most adorable little game to exist.' The official Instagram (@sledding_game) grew to 533,000+ followers. The Demo was available free on Steam until the Early Access launch.
